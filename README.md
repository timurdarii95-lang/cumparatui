Shopping cart


Acest proiect este un exercițiu simplu în Python care folosește dicționare și liste pentru a simula un coș de cumpărături. Programul adaugă produse, calculează totalul și permite modificarea cantității unui produs.
Ce face programul
- Creează un dicționar principal cu:
- numele clientului
- lista de produse
- totalul comenzii
- Adaugă 3 produse în coș
- Calculează totalul (preț × cantitate)
- Afișează produsele folosind items()
- Modifică cantitatea unui produs
- Recalculează totalul
